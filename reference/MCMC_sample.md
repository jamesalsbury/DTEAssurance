# MCMC_sample

An MCMC sample for the example given in Salsbury et al (2024)

## Usage

``` r
MCMC_sample
```

## Format

A data frame with 100000 rows and 1 variables:

- x:

  Sample from the MAP prior

## Source

A MCMC sample for the control group for the example given in
https://onlinelibrary.wiley.com/doi/full/10.1002/sim.10136. Three
historical data seta are used to generate a Meta-Analytic-Predictive
Prior distribution
