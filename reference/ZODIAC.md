# ZODIAC data set

A reconstructed survival data set for the ZODIAC clinical trial

## Usage

``` r
ZODIAC
```

## Format

A data frame with 697 rows and 2 variables:

- Survival time:

  Survival Time (in months)

- Status:

  Event indicator (0=Alive, 1=Dead)

## Source

Reconstructed survival data set from the following publication:
https://www.sciencedirect.com/science/article/abs/pii/S1470204510701327?via%3Dihub
